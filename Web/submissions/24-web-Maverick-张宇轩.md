 # 二、基础指令
- ls	列出目录内容
- cd	切换目录
- pwd	显示当前工作目录的路径
- mkdir	创建新目录
- rmdir	删除空目录
- rm	删除文件或目录
- cp	复制文件或目录
- mv	移动文件或目录，也可用于重命名
- touch	创建空文件或更新文件的时间戳
- find	在文件系统中搜索文件
- cat	连接文件并打印到标准输出
- less	分页查看文件内容，支持向上和向下滚动 
- head	查看文件头部（默认前10行）
- tail	查看文件尾部（默认后10行）
- ping	测试网络连接
- ip addr	显示或配置网络接口信息
- ifconfig	显示或配置网络接口信息
- ssh	远程登录到另一台计算机
- wget	从网络下载文件
